# myNetDisk
 一个简单的网盘项目
 基于Qt的C++项目，实现了网盘的基础功能，包括注册登录、好友系统、私聊群聊、文件操作、分享文件等功能，实现了网络通信、多线程等。
## 实现参考

## 技术相关
编程语言：C++
开发平台：Qt Windows 5.15.2
项目知识点：懒汉模式使用的单例模式,基于QT的网络通信，基于ODBC链接的mysql数据库，socket编程相关，C/S架构
